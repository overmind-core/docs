The `overmind-lab` package is a type-safe TypeScript and JavaScript client for the Overmind REST API. It runs server-side and is generated by [Stainless](https://www.stainless.com/), giving you full TypeScript definitions for every request parameter and response field across all Overmind resources — agents, jobs, datasets, traces, and more.

<Note>
  This is the programmatic REST API library (`overmind-lab`). If you want to instrument your LLM calls and agent runs with distributed tracing, see the [JS/TS Tracing SDK](/docs/tracing/js-sdk) (`@overmind-lab/trace-sdk`) instead.
</Note>

## Installation

While the package is in staging, install it directly from GitHub:

```bash
npm install git+ssh://git@github.com:stainless-sdks/overmind-lab-typescript.git
```

<Note>
  Once the package is published to npm, installation will be: `npm install overmind-lab`
</Note>

## Basic usage

Import the default export, construct a client, and call any method with `await`:

```typescript
import OvermindLab from 'overmind-lab';

const client = new OvermindLab({
  apiKey: process.env['OVERMIND_LAB_API_KEY'],
  environment: 'production', // default
});

const user = await client.auth.retrieveCurrentUser();

console.log(user.id);
```

## TypeScript types

Every request parameter and response field is fully typed. Import the named types directly from `'overmind-lab'` to annotate your own variables:

```typescript
import OvermindLab from 'overmind-lab';

const client = new OvermindLab();

const params: OvermindLab.AgentCreateParams = {
  name: 'my-agent',
  project: '182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e',
  slug: 'my-agent',
};

const agent: OvermindLab.AgentCreateResponse = await client.agents.create(params);

console.log(agent.id);
```

Parameter types follow the pattern `<Resource><Method>Params` (e.g. `AgentCreateParams`, `JobListParams`) and response types follow `<Resource><Method>Response` (e.g. `AgentCreateResponse`, `JobRetrieveResponse`). Hover over any symbol in a modern editor to see inline documentation for every field.

## Authentication

Pass your API key directly to the constructor, or set the `OVERMIND_LAB_API_KEY` environment variable and omit the `apiKey` option — the client picks it up automatically:

```typescript
// Option 1 — explicit key
const client = new OvermindLab({ apiKey: 'my-api-key' });

// Option 2 — environment variable (OVERMIND_LAB_API_KEY)
const client = new OvermindLab();
```

## Error handling

When the client cannot reach the API, or the API returns a 4xx or 5xx response, the SDK throws a subclass of `APIError`. Catch it and inspect `err.status` to branch on the specific error type:

```typescript
import OvermindLab from 'overmind-lab';

const client = new OvermindLab();

const agent = await client.agents
  .create({ name: 'x', project: '182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e', slug: 'slug' })
  .catch(async (err) => {
    if (err instanceof OvermindLab.APIError) {
      console.log(err.status);  // e.g. 400
      console.log(err.name);    // e.g. "BadRequestError"
      console.log(err.headers); // response headers
    } else {
      throw err;
    }
  });
```

The full error class hierarchy and corresponding HTTP status codes:

| Status code | Error class               |
|-------------|---------------------------|
| 400         | `BadRequestError`         |
| 401         | `AuthenticationError`     |
| 403         | `PermissionDeniedError`   |
| 404         | `NotFoundError`           |
| 422         | `UnprocessableEntityError`|
| 429         | `RateLimitError`          |
| ≥ 500       | `InternalServerError`     |
| N/A         | `APIConnectionError`      |

## Retries

The client automatically retries failed requests up to **2 times** with exponential backoff. Retries apply to connection errors, `408 Request Timeout`, `409 Conflict`, `429 Rate Limit`, and any `>=500` server error.

Configure the retry count globally or per-request:

```typescript
// Global default
const client = new OvermindLab({ maxRetries: 0 }); // disable retries

// Per-request override
await client.agents.create(
  { name: 'x', project: '182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e', slug: 'slug' },
  { maxRetries: 5 },
);
```

## Timeouts

Requests time out after **1 minute** by default. Pass `timeout` (in milliseconds) to the constructor or to any individual call:

```typescript
// Global default
const client = new OvermindLab({
  timeout: 20 * 1000, // 20 seconds
});

// Per-request override
await client.agents.create(
  { name: 'x', project: '182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e', slug: 'slug' },
  { timeout: 5 * 1000 },
);
```

On timeout the SDK throws an `APIConnectionTimeoutError`. Timed-out requests are retried by default — set `maxRetries: 0` to disable that behaviour.

## Raw responses

All SDK methods return an `APIPromise`. Use `.asResponse()` to get the raw `fetch` `Response` object as soon as headers arrive, without consuming the response body:

```typescript
const client = new OvermindLab();

// .asResponse() — headers only, body not consumed
const response = await client.agents
  .create({ name: 'x', project: '182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e', slug: 'slug' })
  .asResponse();

console.log(response.headers.get('X-My-Header'));
console.log(response.statusText);
```

Use `.withResponse()` when you need both the parsed data and the raw `Response`:

```typescript
const { data: agent, response: raw } = await client.agents
  .create({ name: 'x', project: '182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e', slug: 'slug' })
  .withResponse();

console.log(raw.headers.get('X-My-Header'));
console.log(agent.id);
```

## Logging

<Note>
  Log messages are intended for debugging only. Their format and content may change between releases.
</Note>

Set the log level via the `OVERMIND_LAB_LOG` environment variable or the `logLevel` client option (the option takes precedence):

```typescript
import OvermindLab from 'overmind-lab';

const client = new OvermindLab({
  logLevel: 'debug', // show all messages
});
```

Available levels, from most to least verbose:

| Level     | What is emitted                           |
|-----------|-------------------------------------------|
| `debug`   | All HTTP requests, responses, and headers |
| `info`    | Info messages, warnings, and errors       |
| `warn`    | Warnings and errors (default)             |
| `error`   | Errors only                               |
| `off`     | Nothing                                   |

At `debug` level, full request and response bodies are logged (authentication headers are redacted). To use a custom logger such as [pino](https://www.npmjs.com/package/pino) or [winston](https://www.npmjs.com/package/winston), pass it as the `logger` option:

```typescript
import OvermindLab from 'overmind-lab';
import pino from 'pino';

const logger = pino();

const client = new OvermindLab({
  logger: logger.child({ name: 'OvermindLab' }),
  logLevel: 'debug',
});
```

## Runtime support

The library requires TypeScript ≥ 4.9 and supports the following runtimes:

- **Node.js** 20 LTS or later
- **Deno** v1.28.0 or higher
- **Bun** 1.0 or later
- **Cloudflare Workers**
- **Vercel Edge Runtime**
- Web browsers (Chrome, Firefox, Safari, Edge)
- Jest 28+ with the `"node"` environment
- Nitro v2.6+

React Native is not supported at this time.

## Advanced

### Custom fetch function

The client expects a global `fetch`. Override it by polyfilling the global or by passing the function to the constructor:

```typescript
import OvermindLab from 'overmind-lab';
import fetch from 'my-fetch';

const client = new OvermindLab({ fetch });
```

### fetchOptions

Pass any `RequestInit`-compatible options through `fetchOptions`. These are applied to every request (request-level options override client-level options):

```typescript
const client = new OvermindLab({
  fetchOptions: {
    // any RequestInit fields
  },
});
```

### Proxy configuration

Use runtime-specific `fetchOptions` to route traffic through a proxy.

**Node.js:**
```typescript
import OvermindLab from 'overmind-lab';
import * as undici from 'undici';

const proxyAgent = new undici.ProxyAgent('http://localhost:8888');
const client = new OvermindLab({
  fetchOptions: { dispatcher: proxyAgent },
});
```

**Bun:**
```typescript
import OvermindLab from 'overmind-lab';

const client = new OvermindLab({
  fetchOptions: { proxy: 'http://localhost:8888' },
});
```

**Deno:**
```typescript
import OvermindLab from 'npm:overmind-lab';

const httpClient = Deno.createHttpClient({ proxy: { url: 'http://localhost:8888' } });
const client = new OvermindLab({
  fetchOptions: { client: httpClient },
});
```
