The `client.traces` resource lets you query LLM traces that Overmind has already collected. Use it to search root-span traces, fetch a complete trace with all its child spans, and list the distinct service names seen across your traces.

<Note>
  The `overmind_lab` client is for **reading** trace data via the REST API. To **send** traces to Overmind from your application — capturing LLM calls, tool calls, and spans in real time — use the tracing SDK (`overmind.init()`). See the [Python Tracing SDK](/docs/tracing/python-sdk) for setup instructions.
</Note>

---

## `traces.list()`

**Signature:** `traces.list(**kwargs) -> TraceListResponse`  
**Maps to:** `GET /api/traces/`

Return one row per trace — specifically the root span (the span with no parent) of each trace the authenticated caller can see. Results are paginated.

<ParamField query="agent" type="str">
  UUID of the agent whose traces you want to list.
</ParamField>

<ParamField query="project" type="str">
  UUID of the project to filter traces by.
</ParamField>

<ParamField query="job" type="str">
  UUID of the optimization job to filter traces by.
</ParamField>

<ParamField query="iteration" type="str">
  UUID of the job iteration to filter traces by.
</ParamField>

<ParamField query="service_name" type="str">
  Filter to traces from this specific service name.
</ParamField>

<ParamField query="service_name_in" type="List[str]">
  Filter to traces from any of these service names. Multiple values may be comma-separated.
</ParamField>

<ParamField query="span_type" type="str">
  Filter by span type. One of: `"llm_call"`, `"tool_call"`.
</ParamField>

<ParamField query="has_error" type="bool">
  When `true`, return only traces that contain at least one error span.
</ParamField>

<ParamField query="trace_id" type="str">
  Filter to the trace with this exact OTel trace ID.
</ParamField>

<ParamField query="span_id" type="str">
  Filter to the trace whose root span has this span ID.
</ParamField>

<ParamField query="name" type="str">
  Filter by root span name.
</ParamField>

<ParamField query="operation" type="str">
  Filter by root span operation string.
</ParamField>

<ParamField query="received_at_gte" type="str | datetime">
  Return only traces received at or after this ISO 8601 timestamp.
</ParamField>

<ParamField query="received_at_lte" type="str | datetime">
  Return only traces received at or before this ISO 8601 timestamp.
</ParamField>

<ParamField query="duration_ns_gte" type="int">
  Return only traces with a root-span duration at or above this value in nanoseconds.
</ParamField>

<ParamField query="duration_ns_lte" type="int">
  Return only traces with a root-span duration at or below this value in nanoseconds.
</ParamField>

<ParamField query="min_duration_ms" type="float">
  Return only traces with a root-span duration at or above this value in milliseconds.
</ParamField>

<ParamField query="max_duration_ms" type="float">
  Return only traces with a root-span duration at or below this value in milliseconds.
</ParamField>

<ParamField query="status_code" type="int">
  Filter by OTel status code of the root span.
</ParamField>

<ParamField query="ordering" type="str">
  Field name to sort results by. Prefix with `-` for descending order.
</ParamField>

<ParamField query="page" type="int">
  Page number within the paginated result set.
</ParamField>

<ParamField query="search" type="str">
  Full-text search term.
</ParamField>

<ResponseField name="count" type="int">
  Total number of traces matching the query.
</ResponseField>

<ResponseField name="results" type="List[Result]">
  Paginated list of root-span trace objects. Each result includes `trace_id`, `span_id`, `name`, `operation`, `service_name`, `span_type`, `duration_ns`, `start_time_ns`, `end_time_ns`, `received_at`, `status_code`, `status_message`, and optional `agent`, `job`, `iteration`, and `project` UUIDs.
</ResponseField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

traces = client.traces.list()
print(traces.count)

# Filter to error traces for a specific agent received in the last day
from datetime import datetime, timedelta, timezone

traces = client.traces.list(
    agent="182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
    has_error=True,
    received_at_gte=datetime.now(timezone.utc) - timedelta(days=1),
    ordering="-received_at",
)
for trace in traces.results:
    print(trace.trace_id, trace.service_name, trace.status_message)
```

## `traces.retrieve()`

**Signature:** `traces.retrieve(trace_id: str) -> TraceRetrieveResponse`  
**Maps to:** `GET /api/traces/{trace_id}/`

Fetch a complete trace by its OTel trace ID — a 32-character hex string. The response includes the root span summary plus every span that shares the same `trace_id`, giving you the full span tree.

<ParamField path="trace_id" type="str" required>
  The OTel 32-character hex trace ID. This is **not** the Overmind row UUID — it is the `trace_id` field returned by `traces.list()`.
</ParamField>

<ResponseField name="trace_id" type="str">
  The OTel trace ID for this trace.
</ResponseField>

<ResponseField name="root" type="Root | null">
  The root span of the trace (the span with `is_root: true` and no `parent_span_id`). Contains the full span fields: `name`, `operation`, `service_name`, `span_type`, `span_id`, `duration_ns`, `start_time_ns`, `end_time_ns`, `attributes`, `events`, `resource_attrs`, `status_code`, `status_message`, and optional `agent`, `job`, `iteration`, `project` UUIDs.
</ResponseField>

<ResponseField name="span_count" type="int">
  Total number of spans in this trace, including the root span.
</ResponseField>

<ResponseField name="spans" type="List[Span]">
  All spans sharing this `trace_id`, each with the same full span field set as `root`. Use `parent_span_id` to reconstruct the span tree.
</ResponseField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

trace = client.traces.retrieve(
    "trace_id",  # 32-char OTel hex trace ID
)
print(trace.trace_id)
print(trace.span_count)

if trace.root:
    print(trace.root.name, trace.root.duration_ns)

for span in trace.spans:
    print(
        span.span_id,
        span.name,
        span.span_type,
        span.parent_span_id,  # None for the root span
    )
```

## `traces.list_services()`

**Signature:** `traces.list_services(**kwargs) -> TraceListServicesResponse`  
**Maps to:** `GET /api/traces/services/`

Return the distinct service names found in traces visible to the authenticated caller. Use this to discover which services have sent traces and to build filter UIs or dashboards. The method accepts the same filter parameters as `traces.list()` so you can scope the service list to a specific agent, project, or time window.

<ParamField query="agent" type="str">
  UUID of the agent to scope the service name query to.
</ParamField>

<ParamField query="project" type="str">
  UUID of the project to scope the service name query to.
</ParamField>

<ParamField query="job" type="str">
  UUID of the job to scope the service name query to.
</ParamField>

<ParamField query="received_at_gte" type="str | datetime">
  Restrict the service list to traces received at or after this ISO 8601 timestamp.
</ParamField>

<ParamField query="received_at_lte" type="str | datetime">
  Restrict the service list to traces received at or before this ISO 8601 timestamp.
</ParamField>

<ParamField query="span_type" type="str">
  Restrict to services that have spans of this type. One of: `"llm_call"`, `"tool_call"`.
</ParamField>

<ParamField query="has_error" type="bool">
  When `true`, restrict to services that have at least one error span.
</ParamField>

<ParamField query="ordering" type="str">
  Field name to sort results by.
</ParamField>

<ParamField query="page" type="int">
  Page number within the paginated result set.
</ParamField>

<ParamField query="search" type="str">
  Search term to filter service names.
</ParamField>

<ResponseField name="count" type="int">
  Total number of distinct service names matching the query.
</ResponseField>

<ResponseField name="results" type="List[str]">
  Paginated list of service name strings.
</ResponseField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

response = client.traces.list_services()
print(response.count)
for service_name in response.results:
    print(service_name)

# Scope to a specific project
response = client.traces.list_services(
    project="182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
)
print(response.results)
```
