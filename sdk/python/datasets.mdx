---

The `client.datasets` resource manages the versioned evaluation datasets that Overmind uses to score agent candidates during optimization jobs. Each call to `datasets.create()` automatically increments the version number for the owning agent, creating an immutable snapshot. Datapoints are never mutated in place — to change the data, you create a new dataset version. You can then promote any version to be the active dataset for its agent by calling `datasets.activate()`.

---

## `datasets.list()`

**Signature:** `datasets.list(**kwargs) -> DatasetListResponse`  
**Maps to:** `GET /api/datasets/`

List all dataset versions visible to the authenticated user. Results are paginated.

<ParamField query="agent" type="str">
  UUID of the agent whose datasets you want to list.
</ParamField>

<ParamField query="generator_model" type="str">
  Filter by the model used to generate this dataset.
</ParamField>

<ParamField query="ordering" type="str">
  Field name to sort results by. Prefix with `-` for descending order.
</ParamField>

<ParamField query="page" type="int">
  Page number within the paginated result set.
</ParamField>

<ParamField query="search" type="str">
  Full-text search term.
</ParamField>

<ParamField query="source" type="str">
  Filter by dataset origin. One of: `"seed"`, `"synthetic"`, `"augmented"`, `"production"`.
</ParamField>

<ParamField query="version" type="int">
  Return only datasets with this exact version number.
</ParamField>

<ParamField query="version_min" type="int">
  Return only datasets at or above this version number.
</ParamField>

<ParamField query="version_max" type="int">
  Return only datasets at or below this version number.
</ParamField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

datasets = client.datasets.list()
print(datasets.count)

# Filter to production datasets for a specific agent
datasets = client.datasets.list(
    agent="182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
    source="production",
)
for ds in datasets.results:
    print(ds.id, ds.version, ds.num_datapoints)
```

## `datasets.create()`

**Signature:** `datasets.create(**kwargs) -> DatasetCreateResponse`  
**Maps to:** `POST /api/datasets/`

Create a new versioned dataset for an agent. Overmind auto-increments the `version` number based on how many datasets already exist for the agent. Pass `make_active=True` to immediately promote this new version as the agent's active dataset.

<ParamField body="agent" type="str" required>
  UUID of the agent this dataset belongs to.
</ParamField>

<ParamField body="datapoints" type="Iterable[Datapoint]" required>
  List of datapoint objects to include in this dataset version. Each datapoint supports: `input` (object), `expected_output` (object), `order` (int), `persona` (str), `tags` (object).
</ParamField>

<ParamField body="name" type="str">
  Optional human-readable name for this dataset version. Up to 255 characters.
</ParamField>

<ParamField body="source" type="str">
  Origin of the dataset data. One of: `"seed"`, `"synthetic"`, `"augmented"`, `"production"`.
</ParamField>

<ParamField body="generator_model" type="str">
  Model used to generate synthetic datapoints, if applicable. Up to 128 characters.
</ParamField>

<ParamField body="make_active" type="bool">
  When `true`, sets this dataset version as the agent's `active_dataset` immediately after creation.
</ParamField>

<ParamField body="metadata" type="object">
  Arbitrary key-value metadata to attach to this dataset version.
</ParamField>

<ParamField body="policy_hash" type="str">
  Hash of the policy in effect when this dataset was created. Up to 64 characters.
</ParamField>

<ResponseField name="id" type="str">
  UUID of the newly created dataset version.
</ResponseField>

<ResponseField name="agent" type="str">
  UUID of the owning agent.
</ResponseField>

<ResponseField name="version" type="int">
  Auto-incremented version number for this agent's dataset sequence.
</ResponseField>

<ResponseField name="num_datapoints" type="int">
  Number of datapoints stored in this dataset version.
</ResponseField>

<ResponseField name="datapoints" type="List[Datapoint]">
  The datapoints as stored, each with `id`, `input`, `expected_output`, `order`, `persona`, and `tags`.
</ResponseField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

dataset = client.datasets.create(
    agent="182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
    datapoints=[
        {
            "input": {"query": "What is the capital of France?"},
            "expected_output": {"answer": "Paris"},
            "order": 0,
        },
        {
            "input": {"query": "What is 2 + 2?"},
            "expected_output": {"answer": "4"},
            "order": 1,
        },
    ],
    name="v2 seed dataset",
    source="seed",
    make_active=True,
)
print(dataset.id)
print(dataset.version)       # auto-incremented
print(dataset.num_datapoints)  # 2
```

## `datasets.retrieve()`

**Signature:** `datasets.retrieve(id: str) -> DatasetRetrieveResponse`  
**Maps to:** `GET /api/datasets/{id}/`

Fetch a single dataset version by UUID. The response includes the full `datapoints` array inline.

<ParamField path="id" type="str" required>
  UUID of the dataset to retrieve.
</ParamField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

dataset = client.datasets.retrieve(
    "182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
)
print(dataset.id)
print(dataset.version)
for dp in dataset.datapoints:
    print(dp.input, dp.expected_output)
```

## `datasets.delete()`

**Signature:** `datasets.delete(id: str)`  
**Maps to:** `DELETE /api/datasets/{id}/`

Permanently delete a dataset version. This action cannot be undone.

<ParamField path="id" type="str" required>
  UUID of the dataset version to delete.
</ParamField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

client.datasets.delete(
    "182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
)
```

## `datasets.activate()`

**Signature:** `datasets.activate(id: str) -> DatasetActivateResponse`  
**Maps to:** `POST /api/datasets/{id}/activate/`

Promote a dataset version to be the **active dataset** for its owning agent. Overmind uses the agent's active dataset when running optimization jobs, so activating a version is the way to switch which data the next job will evaluate against. The response contains the full activated dataset, including its `datapoints` array.

<ParamField path="id" type="str" required>
  UUID of the dataset version to activate.
</ParamField>

<ResponseField name="id" type="str">
  UUID of the now-active dataset.
</ResponseField>

<ResponseField name="agent" type="str">
  UUID of the owning agent, whose `active_dataset` pointer has been updated.
</ResponseField>

<ResponseField name="version" type="int">
  Version number of this dataset.
</ResponseField>

<ResponseField name="num_datapoints" type="int">
  Number of datapoints in this dataset.
</ResponseField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

result = client.datasets.activate(
    "182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
)
print(result.id)
print(result.version)    # version now active for this agent
print(result.agent)      # agent whose active_dataset was updated
```

## `datasets.list_datapoints()`

**Signature:** `datasets.list_datapoints(id: str, **kwargs) -> DatasetListDatapointsResponse`  
**Maps to:** `GET /api/datasets/{id}/datapoints/`

Paginate through the datapoints of a specific dataset version. Prefer this method over `datasets.retrieve()` when the dataset is large, since `retrieve()` returns all datapoints inline while `list_datapoints()` pages them.

<ParamField path="id" type="str" required>
  UUID of the dataset whose datapoints you want to list.
</ParamField>

<ParamField query="agent" type="str">
  Filter datapoints by agent UUID.
</ParamField>

<ParamField query="generator_model" type="str">
  Filter datapoints by the model used to generate them.
</ParamField>

<ParamField query="ordering" type="str">
  Field name to sort results by. Prefix with `-` for descending order.
</ParamField>

<ParamField query="page" type="int">
  Page number within the paginated result set.
</ParamField>

<ParamField query="search" type="str">
  Full-text search term applied to datapoint fields.
</ParamField>

<ParamField query="source" type="str">
  Filter by datapoint origin. One of: `"seed"`, `"synthetic"`, `"augmented"`, `"production"`.
</ParamField>

<ParamField query="version" type="int">
  Filter datapoints by dataset version number.
</ParamField>

<ParamField query="version_min" type="int">
  Filter datapoints from this version onwards.
</ParamField>

<ParamField query="version_max" type="int">
  Filter datapoints up to and including this version.
</ParamField>

<ResponseField name="count" type="int">
  Total number of datapoints matching the filter.
</ResponseField>

<ResponseField name="results" type="List[Result]">
  Paginated list of datapoint objects, each with `id`, `input`, `expected_output`, `order`, `persona`, and `tags`.
</ResponseField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

response = client.datasets.list_datapoints(
    id="182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
    page=1,
)
print(response.count)
for dp in response.results:
    print(dp.id, dp.input)
```
