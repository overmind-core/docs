The `client.agents` resource gives you full programmatic control over Overmind agents. Use it to register new agents, page through existing ones, update configuration, soft-delete agents you no longer need, and fetch the evaluation specification that Overmind uses when running optimization jobs.

---

## `agents.list()`

**Signature:** `agents.list(**kwargs) -> AgentListResponse`  
**Maps to:** `GET /api/agents/`

List all agents visible to the authenticated user. Results are paginated; use `page` to advance through result sets.

<ParamField query="model" type="str">
  Filter agents by the model identifier they are configured to use.
</ParamField>

<ParamField query="name" type="str">
  Filter by agent name (exact or partial match depending on server-side implementation).
</ParamField>

<ParamField query="ordering" type="str">
  Field name to sort results by. Prefix with `-` for descending order (e.g. `-created_at`).
</ParamField>

<ParamField query="page" type="int">
  Page number within the paginated result set. Starts at 1.
</ParamField>

<ParamField query="project" type="str">
  UUID of the project to filter agents by.
</ParamField>

<ParamField query="search" type="str">
  Full-text search term applied across agent name, slug, and description.
</ParamField>

<ParamField query="slug" type="str">
  Filter by the agent's URL-safe identifier slug.
</ParamField>

<ParamField query="status" type="str">
  Filter by agent status string.
</ParamField>

<ResponseField name="count" type="int">
  Total number of agents matching the query.
</ResponseField>

<ResponseField name="results" type="List[Result]">
  Paginated list of agent objects. Each result includes `id`, `name`, `slug`, `project`, `created_at`, `updated_at`, and optional fields such as `model`, `status`, `description`, and `display_name`.
</ResponseField>

<ResponseField name="next" type="str | null">
  URL for the next page of results, or `null` if this is the last page.
</ResponseField>

<ResponseField name="previous" type="str | null">
  URL for the previous page of results, or `null` if this is the first page.
</ResponseField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

agents = client.agents.list()
print(agents.count)

# Filter by project and status
agents = client.agents.list(
    project="182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
    status="active",
    ordering="-created_at",
)
for agent in agents.results:
    print(agent.id, agent.name)
```

## `agents.create()`

**Signature:** `agents.create(**kwargs) -> AgentCreateResponse`  
**Maps to:** `POST /api/agents/`

Register a new agent in Overmind. Three fields are required; all others are optional and can be set at creation time or updated later.

<ParamField body="name" type="str" required>
  Human-readable agent name. Between 1 and 255 characters.
</ParamField>

<ParamField body="project" type="str" required>
  UUID of the project this agent belongs to.
</ParamField>

<ParamField body="slug" type="str" required>
  URL-safe unique identifier for this agent within its project. Between 1 and 255 characters.
</ParamField>

<ParamField body="display_name" type="str">
  Optional display name shown in the Overmind UI. Up to 512 characters.
</ParamField>

<ParamField body="description" type="str">
  Free-text description of what this agent does.
</ParamField>

<ParamField body="agent_path" type="str">
  File path or import path pointing to the agent's entrypoint. Up to 512 characters.
</ParamField>

<ParamField body="analyzer_model" type="str">
  Name of the LLM used by Overmind's analyzer when evaluating this agent. Up to 128 characters.
</ParamField>

<ParamField body="model" type="str">
  Primary model used by the agent itself. Up to 128 characters.
</ParamField>

<ParamField body="status" type="str">
  Lifecycle status string (e.g. `"active"`, `"draft"`). Between 1 and 20 characters.
</ParamField>

<ParamField body="policy_markdown" type="str">
  Markdown-formatted policy document describing agent behavior constraints.
</ParamField>

<ResponseField name="id" type="str">
  UUID assigned to the newly created agent.
</ResponseField>

<ResponseField name="created_at" type="datetime">
  ISO 8601 timestamp of when the agent was created.
</ResponseField>

<ResponseField name="updated_at" type="datetime">
  ISO 8601 timestamp of the most recent update.
</ResponseField>

<ResponseField name="name" type="str">
  The agent name as stored.
</ResponseField>

<ResponseField name="project" type="str">
  UUID of the owning project.
</ResponseField>

<ResponseField name="slug" type="str">
  The agent's URL-safe identifier.
</ResponseField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

agent = client.agents.create(
    name="my-support-agent",
    project="182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
    slug="my-support-agent",
    display_name="Support Agent v1",
    model="gpt-4o",
    analyzer_model="gpt-4o",
    status="active",
)
print(agent.id)
```

## `agents.retrieve()`

**Signature:** `agents.retrieve(id: str) -> AgentRetrieveResponse`  
**Maps to:** `GET /api/agents/{id}/`

Fetch a single agent by its UUID, including all optional metadata fields.

<ParamField path="id" type="str" required>
  UUID of the agent to retrieve.
</ParamField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

agent = client.agents.retrieve(
    "182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
)
print(agent.id)
print(agent.policy_markdown)
```

## `agents.update()`

**Signature:** `agents.update(id: str, **kwargs) -> AgentUpdateResponse`  
**Maps to:** `PATCH /api/agents/{id}/`

Partially update an agent. Only the fields you pass are changed; all others retain their current values.

<ParamField path="id" type="str" required>
  UUID of the agent to update.
</ParamField>

<ParamField body="name" type="str">
  New human-readable name. Between 1 and 255 characters.
</ParamField>

<ParamField body="slug" type="str">
  New URL-safe slug. Between 1 and 255 characters.
</ParamField>

<ParamField body="project" type="str">
  Reassign the agent to a different project UUID.
</ParamField>

<ParamField body="display_name" type="str">
  Updated display name. Up to 512 characters.
</ParamField>

<ParamField body="description" type="str">
  Updated free-text description.
</ParamField>

<ParamField body="agent_path" type="str">
  Updated file or import path. Up to 512 characters.
</ParamField>

<ParamField body="analyzer_model" type="str">
  Updated analyzer model name. Up to 128 characters.
</ParamField>

<ParamField body="model" type="str">
  Updated primary model name. Up to 128 characters.
</ParamField>

<ParamField body="status" type="str">
  Updated status string. Between 1 and 20 characters.
</ParamField>

<ParamField body="policy_markdown" type="str">
  Updated policy Markdown document.
</ParamField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

agent = client.agents.update(
    id="182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
    status="active",
    policy_markdown="## Updated policy\n\nAlways respond concisely.",
)
print(agent.id)
print(agent.updated_at)
```

## `agents.delete()`

**Signature:** `agents.delete(id: str)`  
**Maps to:** `DELETE /api/agents/{id}/`

<Note>
  This is a **soft delete**. The agent is not permanently removed; instead its `is_deleted` flag is set to `true`. The agent record remains accessible via the API with `is_deleted: true` until permanently purged.
</Note>

<ParamField path="id" type="str" required>
  UUID of the agent to soft-delete.
</ParamField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

client.agents.delete(
    "182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
)
```

## `agents.retrieve_eval_spec()`

**Signature:** `agents.retrieve_eval_spec(id: str) -> AgentRetrieveEvalSpecResponse`  
**Maps to:** `GET /api/agents/{id}/eval_spec/`

Retrieve the evaluation specification Overmind uses when scoring candidate agent variants during an optimization job. The spec describes the agent's structure, constraints, and scoring weights.

<ParamField path="id" type="str" required>
  UUID of the agent whose eval spec you want.
</ParamField>

<ResponseField name="agent_description" type="str">
  Textual description of the agent used in evaluation prompts.
</ResponseField>

<ResponseField name="agent_path" type="str">
  Entrypoint path of the agent.
</ResponseField>

<ResponseField name="optimizable_elements" type="object">
  Dictionary of prompt elements that Overmind is allowed to vary during optimization.
</ResponseField>

<ResponseField name="fixed_elements" type="object">
  Dictionary of prompt elements that must not change between iterations.
</ResponseField>

<ResponseField name="total_points" type="float">
  Maximum possible score across all evaluation dimensions.
</ResponseField>

<ResponseField name="structure_weight" type="float">
  Relative weight given to structural compliance in the overall score.
</ResponseField>

<ResponseField name="tool_usage_weight" type="float">
  Relative weight given to correct tool usage in the overall score.
</ResponseField>

```python
from overmind_lab import OvermindLab

client = OvermindLab()

spec = client.agents.retrieve_eval_spec(
    "182bd5e5-6e1a-4fe4-a799-aa6d9a6ab26e",
)
print(spec.agent_description)
print(spec.total_points)
```
