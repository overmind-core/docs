## How it works

Overmind improves agents by running structured optimization loops driven by your own policy and test data.

<Steps>
  <Step title="Register your agent">
    Run `/overmind-register-agent` in Cursor or Claude Code. Overmind reads your code, identifies the entrypoint, and registers the agent in the console.
  </Step>
  <Step title="Generate policy, spec & dataset">
    Run `/overmind-generate-spec-and-dataset`. Overmind infers (or imports) a policy document, builds scoring criteria, and creates a test dataset — all in one pass.
  </Step>
  <Step title="Optimize">
    Run `/overmind-optimize-agent`. Overmind evaluates every test case, diagnoses failures, generates candidate code fixes, and accepts only changes that pass regression checks.
  </Step>
  <Step title="Review results">
    View score history, traces, improvement reports, and diffs at [console.overmindlab.ai](https://console.overmindlab.ai/agents).
  </Step>
</Steps>
